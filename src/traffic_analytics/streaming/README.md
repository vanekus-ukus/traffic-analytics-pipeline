# streaming

Здесь обработка видео и live-режим.

Основные файлы:
- `pipeline.py` - обычный запуск по файлу или ссылке
- `live_runner.py` - segmented live mode
- `yolo_backend.py` - детекция и трекинг
- `video_source.py` - разбор источника
- `storage.py` - запись результатов в БД

Что делает:
- получает видео
- детектит объекты
- собирает треки
- считает оконные метрики
- пишет всё в PostgreSQL
